# Prioritization Matrix — Denial Reduction Initiatives

The following initiatives are prioritized using an **Impact vs. Effort** framework to maximize business value with minimal disruption.

| Initiative | Impact | Effort | Priority | Reason |
|-----------|--------|--------|---------|-------|
| Eligibility Pre-Check | ⭐⭐⭐⭐⭐ | ⭐⭐ | High | Immediate reduction in preventable rejections; easy front-office training |
| Coding Validation Rules | ⭐⭐⭐⭐ | ⭐⭐⭐ | High | Significantly lowers repeat coding denials; moderate configuration/training effort |
| Weekly Denial Analytics Review | ⭐⭐⭐ | ⭐ | Medium | Low effort process change; helps track continuous improvement |
| Automated Payer Feedback Integration | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | Low | High complexity technical solution — future roadmap item |

### Priority Decisions
 Start with **Eligibility Pre-Check** — fastest ROI  
 Deploy **Coding Validation Rules** — medium effort, high reduction in denial rate  
 Add **Weekly Feedback Loop** — ensures sustainability  
 Keep **Automation Enhancements** for Phase-2 implementation
